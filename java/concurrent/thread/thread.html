# 一.Java多线程介绍

# 1.什么是进程？

进程是操作系统中的某个应用程序，每个应用程序都一个单独的进程id

# 2.什么是线程？

线程是应用程序中的多个处理任务，线程是程序执行的最小单元

# 3.线程与进程之间的区别？

线程属于进程，是进程内粒度更小的处理任务
